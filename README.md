# 🎮 Juegos Matemáticos

Una colección de juegos matemáticos interactivos desarrollada como un experimento para una feria de matematicas, exploración de ideas y desarrollo asistido por inteligencia artificial.

## ¿Qué es este proyecto?

Este proyecto reúne distintos juegos y desafíos matemáticos diseñados para hacer que el aprendizaje sea más dinámico e interactivo. Más que un producto final, representa un proceso de construcción, iteración y experimentación utilizando herramientas modernas de desarrollo impulsadas por IA.

Gran parte del desarrollo se realizó utilizando **Cursor**, aplicando un enfoque conocido como **Vibe Coding**: construir, probar, corregir y mejorar rápidamente a través de conversaciones con modelos de inteligencia artificial, manteniendo siempre supervisión humana sobre las decisiones importantes.

## Objetivos

* Hacer las matemáticas más entretenidas mediante juegos interactivos.
* Explorar nuevas formas de desarrollar software asistido por IA.
* Aprender sobre diseño de interfaces, lógica de programación y experiencia de usuario.
* Experimentar con iteraciones rápidas y prototipado moderno.

## Filosofía del Proyecto

Este repositorio no busca demostrar perfección técnica ni seguir una metodología rígida de desarrollo.

Su propósito es demostrar que una idea puede evolucionar rápidamente cuando se combinan:

* Curiosidad.
* Creatividad.
* Herramientas modernas de IA.
* Capacidad de experimentar y aprender mientras se construye.

Muchas funcionalidades nacieron a partir de conversaciones, pruebas rápidas, refactorizaciones y nuevas ideas que surgieron durante el proceso de desarrollo.

## Desarrollo con IA

El proyecto fue construido utilizando un flujo de trabajo centrado en:

* Cursor AI.
* Generación y refactorización asistida por IA.
* Prototipado rápido.
* Iteración continua.
* Vibe Coding.

La IA permitió acelerar tareas repetitivas, generar prototipos funcionales y explorar diferentes enfoques de implementación, mientras que las decisiones finales, validaciones y dirección general del proyecto fueron realizadas manualmente.
